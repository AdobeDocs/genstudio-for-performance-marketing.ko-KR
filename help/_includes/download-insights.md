# 인사이트 다운로드

표 오른쪽 위의 다운로드 (아래쪽 화살표) 아이콘을 클릭하여 볼 수 있는 표에 따라 CSV 파일을 다운로드합니다. 다운로드가 자동으로 시작되고 CSV 파일이 기본 다운로드 위치에 배치됩니다. 일부 표에는 여러 페이지가 있을 수 있으며, 표의 오른쪽 아래에 이 페이지가 표시됩니다. CSV 파일에는 테이블의 _모든_ 페이지의 데이터가 포함되어 있습니다.
